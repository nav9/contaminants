# Contaminants
A simple project I made for my quick reference of contaminants and the use of `d3.js` for displaying relationships. The data mentioned as part of the nodes, the connections and the details, were generated by prompting coding agents (LLM/AI), so do not assume that any of the data is accurate. Always consult a healthcare professional and people specialized in nutrition, to obtain accurate information.
